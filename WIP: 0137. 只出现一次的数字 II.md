# 137. 只出现一次的数字 II

给你一个整数数组 nums ，除某个元素仅出现 一次 外，其余每个元素都恰出现 三次 。请你找出并返回那个只出现了一次的元素。

---

负数？
```js
/**
 * @param {number[]} nums
 * @return {number}
 */
var singleNumber = function(nums) {
  var result = ''
  for (var i = 0; i < 32; i ++) {
    var total = 0
    nums.forEach(num => total += ((num >> i) & 1))
    result = `${total % 3}retult`
  }
  return parseInt(result, 2)
};
```
